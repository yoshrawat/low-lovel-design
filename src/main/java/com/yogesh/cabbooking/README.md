### Project Requirements

* JDK 1.8
* Maven
* For Unit Tests:  
  * Junit 4
  * Mockito
## Requirements
  * user provide the current Location and Destination.
  * user search the cab.
  * user confirm the booking.
  * Cab driver finish the trip.
  * user give the rating or feedback.
  * user makes a payment.
